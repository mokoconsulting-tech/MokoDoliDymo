# MokoDoliDymo for [DOLIBARR ERP & CRM](https://www.dolibarr.org)

## Features

Design and print labels for DYMO LabelWriter thermal label printers directly from Dolibarr. Create label templates using product data, barcodes, addresses, and custom fields from your Dolibarr records.

## Translations

Translations can be completed manually by editing files in the module directories under `langs`.

## Installation

Prerequisites: You must have Dolibarr ERP & CRM software installed (version 19.0 or higher).

### From a GIT repository

Clone the repository in `$dolibarr_main_document_root_alt/mokodolidymo`

```shell
cd ....../custom
git clone https://github.com/mokoconsulting-tech/MokoDoliDymo.git mokodolidymo
```

### Final steps

Using your browser:

  - Log into Dolibarr as a super-administrator
  - Go to "Setup" > "Modules"
  - Find **MokoDoliDymo** under the **Moko Consulting** family and enable it
